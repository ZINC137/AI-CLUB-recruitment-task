# IITG AI Club Recruitment Task – Beyond The Box

This project predicts the **occupation of Atlantis citizens** using demographic and lifestyle data.

- Task: Multi-class classification  
- Model: Random Forest (class-balanced)  
- Evaluation Metric: Macro F1-score  

**Files**
- `AI CLUB task.ipynb` – Model training and prediction
- `submission.csv` – Final predictions
- `atlantis_citizens_final.csv`, `test_atlantis_hidden.csv` – Dataset files
